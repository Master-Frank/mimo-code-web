# 会话与上下文

MiMo Code 的会话系统帮助你管理与 AI 的对话，保持上下文连贯性。

---

## 会话管理

每个会话都是独立的对话上下文。MiMo Code 会自动保存会话历史，你可以随时切换或恢复之前的会话。

### 新建会话

按 `ctrl+x` 然后 `n` 可以创建一个新会话。

### 切换会话

按 `ctrl+x` 然后 `s` 可以查看和切换历史会话。

---

## 上下文窗口

MiMo Code 使用智能上下文管理，确保在有限的 token 窗口中保留最重要的信息。

- 自动压缩长对话历史
- 优先保留最近的关键信息
- 支持手动引用特定文件或代码片段

---

## 文件引用

使用 `@` 符号可以引用项目中的文件，将文件内容添加到上下文中：

```txt
请帮我重构 @src/utils/helpers.ts 中的函数
```

---

## 会话共享

你可以将会话分享给团队成员，方便协作和代码审查。详见[分享会话](/zh/mimocode/share)。
