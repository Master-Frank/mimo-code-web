# 斜杠命令速查

MiMo Code 内置了多种斜杠命令，帮助你快速执行操作。

---

## 通用命令

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `/help` | - | 显示帮助信息 |
| `/exit` | `ctrl+c` | 退出 MiMo Code |
| `/clear` | `ctrl+x` `c` | 清空当前会话 |
| `/reset` | `ctrl+x` `r` | 重置会话 |

---

## 会话管理

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `/new` | `ctrl+x` `n` | 新建会话 |
| `/sessions` | `ctrl+x` `s` | 查看历史会话 |
| `/share` | - | 分享当前会话 |

---

## 模型和工具

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `/models` | - | 切换模型 |
| `/connect` | - | 连接模型提供商 |
| `/agent` | - | 切换 Agent |
| `/skill` | - | 调用 Skill |

---

## 编辑操作

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `/undo` | `ctrl+z` | 撤销上一次修改 |
| `/redo` | `ctrl+y` | 重做上一次修改 |
| `/diff` | - | 查看文件变更 |

---

## 配置和设置

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `/config` | - | 打开配置 |
| `/themes` | - | 切换主题 |
| `/init` | - | 初始化项目 |

---

## 其他命令

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `/compact` | - | 压缩上下文 |
| `/cost` | - | 查看 token 消耗 |
| `/permissions` | - | 管理权限 |
