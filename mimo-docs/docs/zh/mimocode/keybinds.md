# 快捷键

MiMo Code 提供丰富的键盘快捷键，提高你的工作效率。

---

## 通用快捷键

| 快捷键 | 说明 |
|--------|------|
| `ctrl+c` | 中断当前操作 / 退出 |
| `ctrl+d` | 退出 MiMo Code |
| `ctrl+l` | 清空屏幕 |
| `ctrl+z` | 撤销 |

---

## 输入框快捷键

### 光标移动

| 快捷键 | 说明 |
|--------|------|
| `ctrl+a` | 跳到行首 |
| `ctrl+e` | 跳到行尾 |
| `ctrl+f` | 前移一个字符 |
| `ctrl+b` | 后移一个字符 |
| `alt+f` | 前移一个单词 |
| `alt+b` | 后移一个单词 |

### 编辑操作

| 快捷键 | 说明 |
|--------|------|
| `ctrl+k` | 删除到行尾 |
| `ctrl+u` | 删除到行首 |
| `ctrl+w` | 删除前一个单词 |
| `ctrl+y` | 粘贴 |

### 发送消息

| 快捷键 | 说明 |
|--------|------|
| `enter` | 发送消息 |
| `shift+enter` | 换行 |
| `ctrl+j` | 换行 |
| `ctrl+return` | 换行 |

---

## 前导键快捷键

大多数操作使用前导键 `ctrl+x`：先按 `ctrl+x`，再按对应按键。

| 快捷键 | 说明 |
|--------|------|
| `ctrl+x` `n` | 新建会话 |
| `ctrl+x` `s` | 切换会话 |
| `ctrl+x` `c` | 清空会话 |
| `ctrl+x` `r` | 重置会话 |
| `ctrl+x` `z` | 撤销修改 |

---

## 模式切换

| 快捷键 | 说明 |
|--------|------|
| `tab` | 在 Build 和 Plan 模式之间切换 |
| `esc` | 中断当前回合 |

---

## 滚动控制

| 快捷键 | 说明 |
|--------|------|
| `ctrl+up` / `ctrl+k` | 向上滚动 |
| `ctrl+down` / `ctrl+j` | 向下滚动 |
| `page up` | 向上翻页 |
| `page down` | 向下翻页 |
