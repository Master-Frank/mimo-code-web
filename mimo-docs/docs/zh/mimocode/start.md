# 简介

[**MiMo Code**](/zh/index) 是一款面向开发者的 AI 编码代理。它能够理解你的代码库、规划改动、安全地编辑代码，并与你团队已有的工具协同工作。

## 从这里开始

- [快速开始](/zh/mimocode/install) - 从安装到在终端中完成第一次实际修改。
- [连接模型](/zh/mimocode/models-provider) - 配置 API 密钥，接入你想使用的 LLM 提供商。
- [更新日志](/zh/mimocode/changelog) - 及时了解最新功能和改进。

## 你可以用 MiMo Code 做什么

- [理解你的代码](/zh/mimocode/interaction) - 理清仓库的整体结构，并找到合适的切入点。
- [规划并构建功能](/zh/mimocode/modes) - 明确改动范围，使用计划模式更有把握地推动大型任务上线。
- [审查更改](/zh/mimocode/permissions) - 检查 diff、运行检查，并在合并前发现问题。
- [自定义 MiMo Code](/zh/mimocode/config-files) - 使用符合你团队工作方式的规则、技能和命令。
- [使用不同的 Agents](/zh/mimocode/agents) - 配置专门的代理，自定义提示词、模型和工具访问权限。
- [排查与修复](/zh/mimocode/troubleshooting) - 复现问题、定位根本原因，并验证修复效果。
